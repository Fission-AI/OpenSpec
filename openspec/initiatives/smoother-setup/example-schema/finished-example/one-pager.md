# One-pager: Fewer skills on first run

## Problem

A new user runs OpenSpec for the first time and sees a long list of skills and
commands at once. It feels heavy. The first run should feel quick.

## Who it is for

A new user on an existing codebase. See [persona.md](persona.md).

## What we will do

Install a small default set of skills. Keep the full set available for people who
want it.

## Out of scope

Removing any skill. Changing what skills do.

## How we will know it worked

A new user reaches a first win in under a minute.
