# Persona: New user on an existing codebase

> A persona is a short note about a kind of person you build for. It keeps the
> work pointed at a real need.

## Who they are

A developer trying OpenSpec for the first time, on a project that already has a
lot of code.

## What they want

A quick win. To see value in the first minute, without learning ten new
commands.

## What gets in their way

Too many skills and commands at once. Setup steps that do not match the tool
they use.

## How this initiative helps

Fewer default skills, so the first run is short (`simplify-skill-installation`).
Commands that land in the folder their tool expects (`fix-opencode-commands-directory`).
