# Persona: AI coding agent

## Who they are

The AI assistant a developer works with. It reads the plan and helps write the
code.

## What they want

Clear, structured context it can read on its own, without a person pasting it in.

## What gets in their way

Plans kept in formats it cannot read, or spread across many tools.

## How this initiative helps

Specs, changes, decisions, and personas all live in one readable place the agent
can open. When the plan lives in a store, the agent can read it from any repo
that references it.
