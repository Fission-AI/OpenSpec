## 修改后的要求

### 要求：更新行为
更新命令应以团队友好的方式将 OpenSpec 指令文件更新为最新模板。

#### 场景：运行更新命令
- **当** 用户运行 `openspec update` 时
- **则** 用最新模板替换 `openspec/AGENTS.md`

### 要求：文件处理
更新命令应以可预测和安全的方式处理文件更新。

#### 场景：更新文件
- **当** 更新文件时
- **则** 用最新模板完全替换 `openspec/AGENTS.md`

### 要求：核心文件始终更新
更新命令应始终更新核心 OpenSpec 文件并显示 ASCII 安全的成功消息。

#### 场景：成功更新
- **当** 更新成功完成时
- **则** 用最新模板替换 `openspec/AGENTS.md`
